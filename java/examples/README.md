# TWKS Java examples

* [Client](client/)

## Prerequisites

* A Java Development Kit
* Maven
* Installed TWKS dependencies

See the [parent Java documentation](../..) for the above.

## Building

    cd java/examples/client
    mvn package

Substituting other example directories for `client` as desired.

## Running

    cd java/examples/client
    mvn exec:exec

Substituting other example directories for `client` as desired.
